## 项目启动
- Node.js >= 14
- 确保已安装 [yarn](https://yarnpkg.com/)

``` bash
# 克隆项目
git clone https://github.com/Bairuina/graduation-project.git

# 安装依赖
yarn

# 启动项目（账号：bairuina 密码：123456）
yarn start

# 项目打包
yarn build
```
